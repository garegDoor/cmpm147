Description

In this programming assignment, we're building a generator that directly reads inspiring images, renders parameterized images, and adjusts its own designs to optimize an evaluation score. In this case I partnered with Grace Herman, another student and we each challenged eachother to a specific implementation after brainstorming together. The challenge I got was to create an image generator based off of Conway's Game of Life and Noise generation.

Technical

  1. This project integrates inspiring images taken from my camera role, that are selectable from the drop-down on the display page.
  2. The design object is initialized in inspiration-specific ways. Selecting different inspirations changes the image size and therefore how many cells are created in the simulation.
  3. The design is rendered by a combination of Conway's Game of Life and adding Noise dependent on the Mutation Slider.
  4. Also, the rate at which the backgrounds disipates is also dependant on the slider as well as the accuracy of the colors of the cells.

Annealing

While I tried to finetune the way that the Mutation Slider interacts with the simulation to create an "annealing" effect, I still think there was some room for improvement. Annealing the slider does sometimes work, but because of the randomness inherent to the Game of Life the simulation is based off of sometimes it doesn't. To try and improve upon this I tried to finetune the values as I said, but unfortunately after finetuning them for a while, still the best strategy is too set the noise to fill the whole screen with cellular auto-mata and then quickly slide it to 0 to get the long lasting after image with many "alive" cells in the simulation still around.

Reflection

This experiment was really cool and glad I was able to accomplish what I think is something true to the initial challenge.
